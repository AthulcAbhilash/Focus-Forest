# Focus Forest - Study Timer

## Overview
This project was built during my early college phase (S1) as a way to rethink how focus and study time can be tracked.

Instead of using a traditional timer, I wanted to create something that visually represents progress and encourages consistency.

## Features
- Focus session timer
- Visual growth system based on time spent
- Encourages consistency rather than just tracking time
- Simple and clean UI

## Tech Stack
- HTML
- CSS
- JavaScript

## What I Learned
- Handling time-based logic
- Designing user-focused interactions
- Building visual feedback systems
- Translating ideas into interactive UI
